# Certificate Images

This directory contains certificate images for the portfolio.

## Required Files:

1. **oracle-certificate.jpg** - Oracle Certified Foundations Associate certificate
2. **innoverse-hackathon-certificate.jpg** - INNOVERSE 2.0 Hackathon Winner certificate  
3. **innoverse-code-relay-certificate.jpg** - INNOVERSE 2.0 Code Relay Winner certificate

## Instructions:

1. Save your certificate images with the exact filenames listed above
2. Supported formats: JPG, JPEG, PNG
3. Recommended size: 400x300 pixels or larger
4. The images will automatically display in the certificate gallery

## Current Status:

- The certificate gallery is configured to show real images from this directory
- If an image file is missing, a colored placeholder will be shown instead
- Once you add the real certificate files, they will automatically replace the placeholders
